# Deno crontab

## Simple cron for deno server.


### Example:


```ts
import { Cron } from "./cron.ts";

let cron = new Cron();
cron.start();

//job 1
cron.add("* * * * *", () => {
  console.log("aaa");
});

//job 2
cron.add("* * * * *", () => {
  console.log("bbbb");
});

console.table(cron.cronJobs);
```
